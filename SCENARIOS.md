# 🎬 사용 시나리오 예시

## 상황: 새로운 지점 추가하기

### 시나리오
회사에 **"홍대점"**이 새로 오픈했고, 키워드 노출량을 추적하고 싶습니다.

---

## ✨ 해결 방법 (2분 소요)

### 1단계: 구글 시트에 데이터 입력
구글 시트를 열고 새로운 행에 데이터를 입력합니다:

```
날짜   | 매장명  | 플랫폼      | main 키워드    | 총 노출 포스팅 수 | ...
02/08  | 홍대점  | 네이버      | 홍대 맛집      | 15              | ...
02/08  | 홍대점  | 네이버      | 홍대 훠궈      | 8               | ...
02/08  | 홍대점  | 인스타그램  | 홍대 맛집      | 25              | ...
02/08  | 홍대점  | 인스타그램  | 홍대 훠궈      | 18              | ...
```

### 2단계: 대시보드에서 "새로고침" 버튼 클릭
- 대시보드 우측 상단의 "새로고침" 버튼 클릭
- 3초 대기

### 3단계: 완료! 🎉
- **일별 노출량 대시보드**: 홍대점 카드가 자동으로 추가됨
- **전주 대비 증감 대시보드**: 데이터가 쌓이면 자동으로 표시됨

## 💡 추가 시나리오

### 시나리오 1: 키워드 변경
**상황**: "홍대 맛집" 대신 "홍대입구 맛집"으로 키워드를 변경하고 싶다

**해결**:
1. 구글 시트에서 키워드 컬럼 수정
2. 새로고침 버튼 클릭
3. 즉시 반영됨!

---

### 시나리오 2: 매일 아침 데이터 업데이트
**상황**: 매일 아침 9시에 전날 데이터를 입력한다

**작업 흐름**:
```
08:00 - 각 플랫폼에서 키워드 노출량 확인
08:30 - 구글 시트에 데이터 입력
       (날짜: 02/09, 각 지점별 데이터)
09:00 - 대시보드 열기
09:01 - 새로고침 버튼 클릭
09:02 - 전날 데이터 확인 완료!
```

**팁**: 
- 북마크바에 대시보드 추가
- 매일 같은 시간에 체크하는 습관 만들기
- 이상치 발견 시 즉시 대응

---

### 시나리오 3: 월말 리포트 작성
**상황**: 월말에 경영진에게 성과 리포트를 제출해야 한다

**활용 방법**:
1. **전주 대비 증감 대시보드** 열기
2. 각 지점의 증감률 스크린샷
3. 상승 지점 분석:
   - 어떤 키워드가 효과적이었나?
   - 어떤 플랫폼에서 성장했나?
4. 하락 지점 대응 방안:
   - 키워드 전략 수정
   - 컨텐츠 강화 필요

**대시보드에서 확인 가능한 인사이트**:
- ✅ 지점별 성과 한눈에 파악
- ✅ 플랫폼별 효과 비교
- ✅ 키워드별 트렌드 분석
- ✅ 데이터 기반 의사결정

---

### 시나리오 4: 팀원과 공유
**상황**: 마케팅 팀 전체가 대시보드를 확인하고 싶다

**방법 1: 구글 시트 뷰어 권한**
1. 구글 시트 공유 설정
2. 팀원 이메일 추가
3. 권한: "뷰어"

**방법 2: GitHub Pages URL 공유**
1. GitHub Pages로 배포 (QUICKSTART.md 참고)
2. URL을 팀원들에게 공유
3. 북마크하고 언제든지 확인

**방법 3: Slack/Teams 통합**
1. 대시보드 URL을 채널에 고정
2. 매일 아침 "오늘의 키워드 노출량" 확인
3. 이상 징후 발견 시 팀원들과 즉시 논의

---

### 시나리오 5: 지점이 50개로 늘어남
**상황**: 사업 확장으로 지점이 7개 → 50개로 증가

**문제점**:
- 대시보드가 너무 길어짐
- 원하는 지점 찾기 어려움
- 로딩 시간 증가

**해결책**:
1. **SCALING.md**의 "옵션 1: 지점 필터" 적용
2. 지점 선택 버튼 추가
3. 원하는 지점만 선택해서 확인
4. 로딩 시간 대폭 감소

→ **[📊 SCALING.md 참고](SCALING.md)**

## 🎯 실무 팁

### 📅 주간 루틴
**월요일 아침**:
1. 전주 대비 증감 대시보드 확인
2. 급등/급락 키워드 파악
3. 주간 전략 수립

**매일 아침**:
1. 일별 노출량 대시보드 확인
2. 전날 데이터 분석
3. 이슈 발견 시 즉시 대응

### 📊 데이터 분석 예시
**좋은 신호**:
- ✅ 특정 키워드 노출량 급증
- ✅ 신규 키워드 상위 노출
- ✅ 인스타그램 노출 증가

**경고 신호**:
- ⚠️ 주요 키워드 노출량 감소
- ⚠️ 경쟁 키워드 점유율 하락
- ⚠️ 플랫폼별 불균형

### 🚀 성장 전략
1. **높은 성과 지점 벤치마킹**:
   - 어떤 키워드를 사용하는가?
   - 컨텐츠 전략은 무엇인가?
   - 다른 지점에 적용 가능한가?

2. **저성과 지점 개선**:
   - 키워드 전략 재검토
   - 경쟁 분석
   - A/B 테스팅

3. **플랫폼별 전략**:
   - 네이버: 블로그, 카페 최적화
   - 인스타그램: 해시태그, 릴스 활용

## 📞 도움말

### 자주 하는 실수
1. ❌ 구글 시트 공유 설정 안 함
   → ✅ "링크가 있는 모든 사용자" 설정 필수

2. ❌ 날짜 형식 틀림 (2024/02/08)
   → ✅ MM/DD 형식 사용 (02/08)

3. ❌ 플랫폼명 오타 (인스타, Instagram)
   → ✅ "네이버" 또는 "인스타그램" 정확히 입력

4. ❌ 데이터 입력 후 새로고침 안 함
   → ✅ 새로고침 버튼 꼭 클릭!

### 문제 해결
- 데이터 안 보임 → 구글 시트 권한 확인
- 새 지점 안 보임 → 새로고침 버튼 클릭
- 증감률 안 보임 → 최소 14일 데이터 필요

## 🎉 성공 사례

### Case 1: 효율적인 키워드 발굴
**Before**: 매주 수작업으로 엑셀 정리  
**After**: 대시보드로 5분 만에 트렌드 파악  
**결과**: 주간 분석 시간 90% 감소

**After**: 증감률 데이터로 전략 수립  
**결과**: 노출량 평균 35% 증가

### Case 3: 팀 협업 강화
**Before**: 개인별로 따로 데이터 관리  
**After**: 하나의 대시보드로 통합 관리  
**결과**: 커뮤니케이션 효율 향상
